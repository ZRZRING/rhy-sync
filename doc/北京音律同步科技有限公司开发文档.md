### 获取歌单列表

- GET /songlist
- Params: pageNum: string, pageSize: string, search: boolean
- Response: { code: string, msg: string, data: { records: SongList[], total: number } }
- SongList: { id: number, title: string, pic: string, introduction: string, style: string }

### 获取歌单中的歌曲

- Params: id: string
- Response: { code: string, msg: string, data: { records: Song[] }, total: number, size: number, current: number, pages: number }
